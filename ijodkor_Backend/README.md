# 🎨 Maktab Market — O'quvchilar ijodi do'koni

## Texnologiyalar
- React.js 19 + Vite 8
- Tailwind CSS v4
- Zustand (state management)
- React Router v7
- vite-plugin-pwa (mobil o'rnatish)
- React Hot Toast + Lucide React

## O'rnatish

```bash
cd school-market
npm install
npm run dev        # development
npm run build      # production build
npm run preview    # build preview
```

Brauzer: http://localhost:5173
